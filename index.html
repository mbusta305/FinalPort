Portfolio

AirMail App

A web application developed for kiosks, that would allow airline passengers who get pulled out of line in security check-point areas for having an item that is not allowed on an airplane, the option to mail these items home instead of throwing it away. I integrated the user authentication using bCrypt, implemented the Easy-Post API to render shipping forms, cost, and shipping labels, as well as integrated the Stripe API allowing users to safely and securely send their credit card information to pay for shipping. Alongside the back-end development, I developed the front-end using HTML, CSS, and JavaScript to style the pages.

N-Lite App

A web application that allows users to find web developers from Wyncode. It shows a link to the developers portfolio, comments about the developers and even allows users to upvote or downvote the developers. The back end of the app is built on Ruby on Rails along with Haml and Sass for the front-end styling.
